USB Rubber Ducky Defender

Overview: This project involves an ML model to detect malicious USB devices (like Rubber Ducky) with 99.85% accuracy. It analyzes USB packet data in real-time to prevent unauthorized keystroke injections and data exfiltration.

Tech Stack: Python, RandomForestClassifier, Scikit-learn, Scapy for packet analysis.

Key Features:

  • Feature extraction from USB HID reports and packet metadata.
  • Trained on a custom dataset of 10K samples (benign vs. malicious USB behaviors).
  • Deployment: Integrated with a Streamlit dashboard for real-time monitoring.
  • Challenges Overcome: Handling noisy packet data and achieving low false positives (0.15%).

Results: Reduced detection time to under 1 second; tested on Kali Linux with simulated attacks.

Future Enhancements: Add deep learning for anomaly detection and support for more USB protocols.
